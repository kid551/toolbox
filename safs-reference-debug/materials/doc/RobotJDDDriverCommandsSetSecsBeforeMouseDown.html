RobotJ DDDriverCommands::SetSecsBeforeMouseDown

RJ
Set the delay before mouse down.

Set the delay before mouse down. This time is seconds.

Fields: [ ]=Optional with Default Value

  1. Value
    The value of the delay before mouse down.

    The value of the delay before mouse down.

Examples:

  • C, SetSecsBeforeMouseDown, 0.1
    Set the delay before mouse down to 0.1 second.
