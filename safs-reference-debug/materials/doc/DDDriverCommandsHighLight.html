DDDriverCommands::HighLight

TC SE2
Turn on or off the component highlight functionality.

When enabled (ON) each call to a component function will temporarily highlight the targeted component. The intent is to provide a design-time verification that the tool is finding the desired target component. It is not recommended this command be used during production runtime testing as there can be significant performance impact with some tools.

Fields: [ ]=Optional with Default Value

  1. [ Mode = ]
    The status for highlight functionality. It can be On or Off. The Default is OFF.


Examples:

  • C, HighLight, ON
    Turn on the highlight functionality.

  • C, HighLight, OFF
    Turn off the highlight functionality.
