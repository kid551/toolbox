Selenium DDDriverCounterCommands::StopCycle
TID

Stop tracking status on a user-defined item.

Log/process the end of testing associated with a user-defined item. A private counter will keep track of status for the item.

Fields: [ ]=Optional with Default Value
  1. CycleID
    The user-defined ID of the item.


  2. [ Description = ]
    A description of the item.


Examples:
  • C StopCycle Option1
    Logs the end of testing on an item with an ID of Option1.

    The private status counter is also stopped.

  • C StopCycle Option1 "Integration Testing"
    Logs the end of testing on an item with an ID of Option1.

    The private status counter is also stopped.
