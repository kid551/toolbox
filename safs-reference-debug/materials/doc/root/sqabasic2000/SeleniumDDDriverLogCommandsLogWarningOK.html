Selenium DDDriverLogCommands::LogWarningOK
TID

Log an EXPECTED WARNING message.

Log that a WARNING was expected and, as such, is OK.
You can optionally add an additional details description that follows the message.

Fields: [ ]=Optional with Default Value
  1. Message
    The message to log.


  2. [ Details = ]
    Additional detail to add to the message.


Examples:
  • C LogWarningOK "No value specified."
    An expected WARNING message logged with no additional detail.


  • C LogWarningOK "No value specified" "Using default value."
    An expected WARNING message logged with additional detail.
