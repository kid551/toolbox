Selenium DDDriverCommands::StopWebBrowser
SE2

Stop an opened web browser indicated by ID.

Stop an opened web browser indicated by ID. This ID is assigned by user during calling keyword StartWebBrowser.

Fields: [ ]=Optional with Default Value
  1. [ BrowserID = MainApp ]
    The indentifier used to get the webdriver for the opened browser. This ID is assigned by user during calling keyword StartWebBrowser.


Examples:
  • C, StopWebBrowser
    Stop an opened browser indicated by default ID 'MainApp'.


  • C, StopWebBrowser, "FirstTestPage"
    Stop an opened browser indicated by ID 'FirstTestPage'.
