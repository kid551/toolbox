Autoit DDDriverLogCommands::LogTestSuccess
TID

Log a SUCCESS/OK message. Also increments appropriate counters.

Log a SUCCESS/OK message. Also increments appropriate counters.
You can optionally add an additional details description that follows the message.

Fields: [ ]=Optional with Default Value
  1. Message
    The message to log.


  2. [ Details = ]
    Additional detail to add to the message.


Examples:
  • C LogTestSuccess "Test 1 Passed"
    A test success message logged with no additional detail


  • C LogTestSuccess "Test 1 Passed" "Test 1 is good"
    A test success message logged with detail added
