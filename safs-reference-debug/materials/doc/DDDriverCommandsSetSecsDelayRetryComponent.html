DDDriverCommands::SetSecsDelayRetryComponent

RJ
Set the delay of re-finding object.

Set the delay of re-finding object. This time is seconds.

Fields: [ ]=Optional with Default Value

  1. Value
    The value of the delay of re-finding object.

    The value of the delay of re-finding object.

Examples:

  • C, SetSecsDelayRetryComponent, 1
    Set the delay of re-finding object to 1 second.
