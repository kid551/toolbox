RobotJ DDDriverCommands::GetCompScreenResolution

RJ
Gets the current computer screen resolution and returns x and y values.

Gets the current computer screen resolution and returns x and y values.

Fields: [ ]=Optional with Default Value

  1. XWidthVariable
    SAFSVARS name to be used to store x screen resolution

  2. YHeightVariable
    SAFSVARS name to be used to store y screen resolution

Examples:

  • C, GetCompScreenResolution, xres, yres
    Gets the current computer screen resolution and stores x in xres and y in yres
