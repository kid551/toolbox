DDDriverCommands::SetSecsSeekComponent

RJ
Set the timeout of finding object.

Set the timeout of finding object. This time is seconds.

Fields: [ ]=Optional with Default Value

  1. VariableName
    The value of the timeout of finding object.

    The value of the timeout of finding object.

Examples:

  • C, SetSecsSeekComponent, 1
    Set the timeout of finding object to 1 second.
