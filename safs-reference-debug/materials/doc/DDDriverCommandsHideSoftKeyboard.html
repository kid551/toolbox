DDDriverCommands::HideSoftKeyboard

DRD
Hide the soft keyboard.

Hide the soft keyboard. The soft keyboard will be shown if user click on an edit box, but the soft keyboard will show over the other components. User can use this keyword to make this soft keyboard disappear.

Fields: [ ]=Optional with Default Value

    Examples:

    • C, HideSoftKeyboard
      Hide the soft keyboard.
