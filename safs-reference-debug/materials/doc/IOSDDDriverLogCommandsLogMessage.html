IOS DDDriverLogCommands::LogMessage

TID
Log a generic message

Log a generic message. You can optionally add an additional details description that follows the message.

Fields: [ ]=Optional with Default Value

  1. Message
    The message to log.

  2. [ Details = ]
    Additional detail to add to the message.

Examples:

  • C LogMessage "Starting Test 1"
    A message logged with no additional detail

  • C LogMessage "Starting Test 1" "Start Login Tests"
    A message logged with detail added
