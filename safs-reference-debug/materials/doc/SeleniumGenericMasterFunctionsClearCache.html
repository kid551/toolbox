Selenium GenericMasterFunctions::ClearCache

SE2
Clear the cache of a component.

Clear the cache of a component. To save time, component may contain a cache to store its content or some time-consuming resource. But sometime, the cache's content may not be fresh, which will affect the test, at this situation, user may call this keyword to clear the cache and SAFS will use the latest content.
NOTE:
SE2 : Supported For Selenium WebDriver.

Fields: [ ]=Optional with Default Value

    Examples:

    • T, WINDOW, COMPONENT, ClearCache
      Clear the current component's cache.
