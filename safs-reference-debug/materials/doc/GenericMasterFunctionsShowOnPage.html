GenericMasterFunctions::ShowOnPage

SE2
Make the component visible on the page.

Make the component visible on the page.

Fields: [ ]=Optional with Default Value

  1. [ verify = False ]
    "True" to verify that the component is shown on page.

    This parameter is boolean, default value is false. Verify that the component is shown on page if this parameter is true.

Examples:

  • T, Window, Comp
    Show the component Comp on the page, make it visible.

  • T, Window, Comp, True
    Show the component Comp on the page, make it visible. And verify that this component is shown.
