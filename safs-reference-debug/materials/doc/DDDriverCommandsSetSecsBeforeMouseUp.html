DDDriverCommands::SetSecsBeforeMouseUp

RJ
Set the delay before mouse up.

Set the delay before mouse up. This time is seconds.

Fields: [ ]=Optional with Default Value

  1. Value
    The value of the delay before mouse up.

    The value of the delay before mouse up.

Examples:

  • C, SetSecsBeforeMouseUp, 0.1
    Set the delay before mouse up to 0.1 second.
